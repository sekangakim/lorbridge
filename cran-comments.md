## Comments to CRAN

This is the first submission of `lorbridge` (v0.1.0).

The package provides a unified analytical workflow bridging binary and
multinomial logistic regression with singly-ordered (SONSCA) and
doubly-ordered (DONSCA) nonsymmetric correspondence analysis.
Log-odds ratios from logistic regression are re-expressed as cosine
theta and closeness-of-concordance measures (CCMs) on the familiar
[-1, +1] scale (Kim & Grochowalski, 2019
<doi:10.3758/s13428-018-1161-1>), with bootstrap confidence intervals
throughout.

The package includes four datasets, 13 exported functions, a vignette,
and a testthat test suite. All examples run in well under 5 seconds.
